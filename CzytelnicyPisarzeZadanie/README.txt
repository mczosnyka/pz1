OPIS:
Program implementuje problem czytelników i pisarzy korzystając z dwóch semaforów a także ze zmiennych opisujacych, czy w czytelni jest pisarz czy czytelnicy.



IN ORDER TO LAUNCH THE PROGRAM:
1. Open CMD/Bash/PowerShell
2. Go to "CzytelnicyPisarzeZadanie" directory
3. Type "mvn clean package"
4. Go to CzytelnicyPisarzeZadanie/main/target
5. Type java -jar CzytelnicyPisarzeZadanie-1.0-SNAPSHOT-jar-with-dependecies
6. This should open the program!

FILES & DIRECTORIES:
main - directory containing the program and the tests
index.html - javadoc.
sonarqube.pdf - pdf file. It's a sonarqube raport

other files are javadoc files or files that are required for the program to work properly
